# Shibili Aman - Portfolio

The official portfolio of Shibili Aman TK, a Full-Stack Developer and B.Tech CSE student. This project is built using Astro v7, React v19, and Tailwind CSS v4, featuring high-performance rendering, a dynamic 3D interactive particle field, and a luxury brutalist theme.

## 🚀 Tech Stack

- **Framework:** Astro 7
- **UI Library:** React 19
- **Styling:** Tailwind CSS 4
- **Animations:** GSAP & Lenis (Smooth Scroll)
- **3D Engine:** Three.js / React Three Fiber
- **Deployment:** Vercel (Static Platform)

## 📂 Project Structure

- `/src/components`: Reusable UI components including the moving screen frame, cursor, and interactive canvas.
- `/src/layouts`: Core page layouts and metadata wrapper.
- `/src/pages`: Router pages for home, projects, work experience, and contact.
- `/src/data`: Portfolio data files.
- `/public`: Static assets including icons, robots.txt, and sitemaps.

## 🛠️ Local Development

1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Start the local server:
   ```bash
   pnpm dev
   ```
3. Build for production:
   ```bash
   pnpm build
   ```
